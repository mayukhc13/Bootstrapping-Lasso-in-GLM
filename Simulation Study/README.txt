This folder contains reproducible codes in R separately for Logistic , Gamma and Linear Regression setup.
Change (n,B)=(sample size, Bootstrap iteration count) as required. 
(p,p_0) is taken as (7,4) fixed throughout.
The entire data set is generated simulation=500 times.
The design matrix is kept fixed.
CVXR package for convex optimization is used, glmnet package is for elastic net in Logistic and Linear Regression. 
h2o package is used for elastic net in Gamma regression.
Run these .R files as required.
